===README===
LOBSTER AGENT README

LOBSTER AGENT는 Streamlit을 기반으로 개발된 AI 어드바이저입니다. 
LOBSTER AGENT는 사용자의 지시를 받아서 AI 모델을 통해 필요한 툴을 자동으로 실행하여 결과물을 생성합니다.

LOBSTER AGENT는 다음과 같은 기능을 제공합니다:

- 무제한 웹검색 (무료)
- 웹페이지 읽기 (무료)
- 로컬 파일 제어 (무료)
- 파이썬 터미널 실행 (무료)
- 숏폼 영상 제작기 (무료)
- SNS 자동 업로드 (웹훅)
- Notion API
- Slack API

LOBSTER AGENT는 Streamlit Secrets에 API 키를 세팅해야 합니다.

LOBSTER AGENT는 다음과 같은 환경 변수를 필요로 합니다:

- GROQ_API_KEY
- GEMINI_API_KEY
- DISCORD_WEBHOOK_URL

LOBSTER AGENT는 다음과 같은 모듈을 필요로 합니다:

- groq
- streamlit
- json
- tools

LOBSTER AGENT는 다음과 같은 함수를 제공합니다:

- LobsterAgent.__init__
- LobsterAgent.execute_tools
- LobsterAgent.think_and_act

LOBSTER AGENT는 다음과 같은 클래스를 제공합니다:

- LobsterAgent
